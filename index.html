História dos computadores

A evolução dos computadores passou por gerações marcadas por inovações de hardware (válvulas, transistores, circuitos integrados) e pelo avanço de software e redes. Um bom panorama pode ser encontrado na história do computador.

Em resumo, saímos de máquinas enormes e especializadas para dispositivos pessoais conectados, com poder de processamento acessível e uso massivo da internet. Voltar ao índice.

O que são Clients?

Client (ou cliente) é o lado que consome um serviço — por exemplo, o navegador que renderiza HTML, CSS e JavaScript. Ele envia requisições ao servidor e apresenta as respostas ao usuário. Veja também: como a Web funciona (MDN).

Voltar ao índice.

O que são Servers?

Server (ou servidor) é a aplicação/infraestrutura que fornece recursos: páginas, arquivos e APIs. Ele recebe requisições dos clientes, processa regras de negócio e devolve respostas (ex.: HTML ou JSON).

Estrutura básica de um HTML

Um documento HTML começa com <!DOCTYPE html>, seguido de <html> (raiz), <head> (metadados) e <body> (conteúdo visível). Títulos usam <h1..h6>, parágrafos <p> e links <a>.

Listas ordenadas vs. não ordenadas

Use lista ordenada (<ol>) quando a sequência importa (passo a passo), e lista não ordenada (<ul>) para coleções sem ordem específica.

Exemplo de lista ordenada

  1. Abrir o editor
  2. Criar o arquivo index.html
  3. Adicionar a estrutura mínima

Exemplo de lista não ordenada

  • HTML
  • CSS
  • JavaScript
